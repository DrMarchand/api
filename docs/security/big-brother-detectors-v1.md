# Big Brother Detectors v1

Date Established: 2026-06-15
Originating Author: Joseph Kyle Marchand
Business Root: Design Orchard LLC

## Purpose

Big Brother Detectors provide read-only visibility for suspicious account, storage, Git, SSH, and secret-file patterns.

## Pirate Flag Rule

A pirate flag means suspicious, unknown, unregistered, or high-risk visibility event.

It does not mean confirmed compromise.

## Current Detectors

- unknown cloud mounts
- unregistered Google Drive accounts
- possible secret filenames
- SSH sensitive key presence
- Git remote/status visibility
- disk pressure
